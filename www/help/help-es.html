Concepto

Maniana es una aplicación para administrar listas de tareas, de manera sencilla y divertida. Maniana te provee de dos páginas, una al lado de la otra:
  • HOY Página para las tareas de hoy.
  • MAÑANA para las Tareas que postergo.
﻿Cada mañana encontrarás todas las tareas en la página HOY. Puedes postergar las tareas que no vas a hacer hoy enviándolas a la página MAÑANA, solo haciendo clic en las flechas.

Deslizar páginas:

Deslizar la pantalla a derecha o izquierda para alternar entre las páginas de HOY y MAÑANA.

Widgets

Mañana incluye dos tipos de widgets que se pueden incrustar en la pantalla de inicio de tu teléfono:
  • Widgets de Lista - muestra las tareas activas de HOY. Se pueden seleccionar cuatro tamaños diferentes.
  • Widget Icono - muestra el número de tareas activas para HOY.
Tocando un widget, se abre la página de HOY. Puede cambiar la apariencia del widget Lista en la pantalla de configuración de Mañana (ver más abajo).

La barra de herramientas en la parte superior del widget contiene iconos de acceso directo para agregar nuevas tareas mediante el teclado o por voz. Puede desactivar la barra de herramientas en las opciones de configuración.

Ajustes

SONIDO

  • Sonido - alterna entre sonido encendido y apagado.
  • Aplausos - controla los aplausos cuando una tarea se ha completado.

COMPORTAMIENTO

  • Animación de inicio - animación de desplazamiento de página cuando se inicia la aplicación.
  • Mensajes detallados - habilita mensajes de respuesta detallados (para usuarios nuevos).
  • Notificación diaria - al habilitarla, Mañana genera una notificación de Android diaria de las tareas activas (si hubiere).
  • Notificación por LED - si esta marcada, además de las notificaciones diarias, también se notifica con el dispositivo LED. (si está disponible).
  • Calendario Google - si esta marcada, al hacer clic en la fecha de hoy, abre el Calendario de Google (App Google Calendar).
  • Acción de sacudido - establece una acción predeterminada, en respuesta al evento de sacudir el dispositivo.

TAREAS

  • Orden Automático de Tareas - si esta activada, las tareas se ordenan automáticamente: las activas antes de las completadas.
  • Añadir al principio - determina si se agregan nuevas tareas en la parte superior o inferior de la lista.
  • Colores de Tarea - selecciona y ordena los colores de tarea disponibles.
  • Limpieza de tareas diarias - si está activada, cada medianoche, todas las tareas completadas se borran automáticamente.
  • Bloqueo de Caducidad - establece bloqueos caducidad automática (semanal, mensual o nunca).

PÁGINAS

  • Establecer por Tema - permite seleccionar una de las apariencias de página predefinidas. Igualmente, puede utilizar la configuración para personalizar el Tema.
  • Papel - utilizar el papel como fondo, si esta desactivado, debe configurar el color de fondo sólido.
  • Color de fondo - selecciona el color de fondo o de papel respectivamente (depende de la opción papel). El color de fondo puede ser semi transparente si así lo desea.
  • Iconos - Selecciona el conjunto de iconos.
  • Color del Separador de tareas - selecciona el color de las líneas entre tareas (puede hacerse transparente).
  • Texto del título - selecciona la tipografía, el tamaño y el color del texto de título de página.
  • Mostrar fecha - si está marcada, en la barra de título del Widget de lista se muestra la fecha actual en lugar de 'Hoy'.
  • Texto de tarea - establece la tipografía, tamaño y color del texto de tareas.

WIDGET DE LISTA

  • Establecer por Tema - permite seleccionar una de las apariencias de página predefinidas. Igualmente, puede utilizar la configuración para personalizar el Tema.
  • Papel - Utilizar fondo de papel, si esta desactivado, debe configurar el color de fondo sólido.
  • Color de fondo - selecciona el color de fondo o de papel respectivamente (depende de la opción papel). El color de fondo puede ser semi transparente si así lo desea.
  • Mostrar la barra de herramientas - si se activa, la barra de herramientas con los iconos de acceso directo se muestra en la parte superior del widget de lista.
  • Texto de tarea - selecciona color, tamaño y fuente del texto de tarea.
