Концепция

Maniana — это простое приложение для управления списком задач. Оно состоит из двух страниц:
  • СЕГОДНЯ: страница для задач на сегодня.
  • MANIANA: страница для будущих заданий.
Каждое утро вы найдете все свои задачи на странице «СЕГОДНЯ». Те задачи, которые вы планируете отложить на потом, можно отправить на страницу «MANIANA», нажав на стрелки справа от них.

Переключение страниц:

Проведите пальцем слева направо или справа налево, чтобы переключиться между страницами «СЕГОДНЯ» и «MANIANA».

Опции меню задач

Нажмите на задачу, чтобы открыть всплывающее меню:

Переключение статуса задачи (выполнено/не выполнено).
Измените текст задачи.
Заблокируйте/разблокируйте задачи. Заблокированные задачи остаются на странице «MANIANA» до истечения срока блокировки.
Удалите задачу (возможна отмена действия).

Цвет задач

Нажмите на самый левый край задачи для переключения цвета.

Порядок задач

Удерживая палец на задаче, перетащите её вверх или вниз.

Виджеты

Maniana использует два типа виджетов, которые вы можете разместить на домашнем экране вашего устройства:
  • Виджеты списка — показывают активные задачи со страницы «СЕГОДНЯ». Существуют виджеты четырёх различных размеров.
  • Виджет значка — Показывает число активных задач на странице «СЕГОДНЯ».
При нажатии на виджет открывается страница «СЕГОДНЯ». Внешний вид виджетов списка можно настроить на экране параметров Maniana (см. ниже).

Панель инструментов в верхней части виджета содержит значки для добавления новых задач при помощи клавиатуры или голоса. Её можно отключить в настройках.

Настройки

ЗВУК

  • Звук — включает или выключает звук.
  • Аплодисменты — включает или выключает аплодисменты при выполнении задачи.

ПОВЕДЕНИЕ

  • Анимация при запуске — показывает при запуске Maniana анимированное перелистывание страниц.
  • Подробные сообщения — включает более подробные сообщения (для новых пользователей).
  • Ежедневные уведомления — если выбрано, Maniana будет ежедневно выводить стандартные уведомления Android при наличии текущих задач.
  • LED-уведомления — если включено, ежедневные уведомления будет также активировать LED-индикатор (если таковой имеется).
  • Календарь Google — если включено, нажатие на сегодняшнюю дату будет запускать календарь Google.
  • Действие при встряхивании — задает действие страницы Maniana, выполяемое при встряхивании устройства.

ЗАДАЧИ

  • Автоматическая сортировка задач — если этот флажок установлен, текущие задачи всегда будут отображаться в списке над завершенными и заблокированными задачами.
  • Добавлять в начало списка — определяет, будут новые задачи добавляются в верхнюю или нижнюю часть списка.
  • Цвета задач — позволяет выбрать и расположить в нужном порядке цвета задач.
  • Ежедневная очистка списка задач — если флажок установлен, каждый вечер все завершённые задачи будут удаляться.
  • Срок действия блокировки — устанавливает срок автоматического истечения блокировки (через неделю, месяц или никогда).

СТРАНИЦЫ

  • Выбрать тему — нажмите, чтобы выбрать одну из предложенных тем. Её можно дополнительно настроить с помощью параметров ниже.
  • Бумага — поставьте галочку, чтобы использовать бумажный фон, или снимите её, если вам нужен одноцветный фон.
  • Цвет бумаги/фона — задаёт цвет бумаги или фона соответственно (в зависимости от того, что выбрано в пункте «Бумага») При желании цвет фона можно сделать полупрозрачным.
  • Значки — задаёт стиль значков на странице.
  • Цвет разделителя задач — задаёт цвет линий между задачами (может быть прозрачным).
  • Текст заголовка — задаёт шрифт, размер и цвет текста заголовка страницы.
  • Показывать дату — если включено, заголовок виджета списка будет показывать текущую дату вместо слова «СЕГОДНЯ».
  • Текст задачи — задаёт шрифт, размер и цвет текста задачи.

ВИДЖЕТ СПИСКА

  • Выбрать тему — нажмите, чтобы выбрать одну из предложенных тем. Её можно дополнительно настроить с помощью параметров ниже.
  • Бумага — поставьте галочку, чтобы использовать бумажный фон, или снимите её, если вам нужен одноцветный фон.
  • Цвет бумаги/фона — задаёт цвет бумаги или фона соответственно (в зависимости от того, что выбрано в пункте «Бумага»). При желании цвет фона можно сделать полупрозрачным.
  • Показывать панель инструментов — если включено, в верхней части виджета списка будет отображаться панель инструментов.
  • Текст задач — задаёт шрифт, размер и цвет текста задач.

РЕЗЕРВНЫЕ КОПИИ (ЭКСПЕРИМЕНТАЛЬНО)

ДОПОЛНИТЕЛЬНО

  • Сведения о релизе — нажмите, чтобы увидеть номер версии и список изменений.
  • Рассказать другу — вам нравится Maniana? Нажмите, чтобы рассказать об этом приложении другу.
  • Отправить отзыв — нажмите, чтобы отправить разработчикам Maniana свой отзыв, вопрос или предложение.
  • Сбросить все настройки — щелкните для восстановления параметров по умолчанию (после запроса на подтверждение). Данное действие не отражается на данных текущих задач.
